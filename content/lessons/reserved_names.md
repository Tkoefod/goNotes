## 25 keywords that are reserved

Go has 25 keywords that are reserved for the system.

- break
- select
- case
- struct
- chan
- switch
- const
- type
- continue
- var
- default
- defer
- else
- fallthrough
- for
- func
- go
- goto
- if
- import
- interface
- map
- package
- range
- return
